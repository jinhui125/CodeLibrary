16. 使用媒体查询实现响应式布局

/* 默认样式 - 移动设备优先 */ .container { width: 100%; padding: 15px; margin: 0 auto; } .sidebar { display: none; } /* 平板设备 */ @media (min-width: 768px) { .container { max-width: 720px; } .main-content { display: grid; grid-template-columns: 2fr 1fr; gap: 20px; } .sidebar { display: block; } } /* 桌面设备 */ @media (min-width: 1024px) { .container { max-width: 960px; } .main-content { grid-template-columns: 3fr 1fr; } } /* 大桌面设备 */ @media (min-width: 1200px) { .container { max-width: 1140px; } }

其他题目答案思路

以下是部分未展示完整答案的题目实现思路：

  • 13. 实现圣杯布局：使用Grid或Flexbox创建页眉、页脚和三栏布局，中间为主内容，左右为边栏
  • 14. 使用定位实现叠加效果：使用position: relative和position: absolute将元素叠加在另一个元素上
  • 15. 创建粘性页脚：使用Flexbox或Grid布局，将页脚固定在容器底部
  • 17. 创建响应式导航菜单：使用媒体查询在移动设备上隐藏导航项，显示汉堡菜单
  • 18. 实现响应式图像：使用picture元素或srcset属性为不同屏幕提供不同尺寸的图像
  • 19. 使用相对单位：使用rem、em和%等相对单位替代固定像素值
  • 20. 创建响应式表格：使用overflow-x: auto使表格在小屏幕上可水平滚动
  • 23. 实现渐变背景：使用linear-gradient或radial-gradient创建渐变背景
  • 24. 创建阴影效果：使用box-shadow和text-shadow属性创建阴影效果
  • 25. 使用变换属性：使用transform属性实现旋转、缩放、倾斜和平移效果
