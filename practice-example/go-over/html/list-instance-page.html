代码用例【有序列表】：


  <ol type="A">
      <li>啦啦啦</li>
      <li>叭叭叭</li>
      <li>嘻嘻嘻</li>
  </ol>

  1. 啦啦啦
  2. 叭叭叭
  3. 嘻嘻嘻

      <li>嘻嘻嘻</li>
  </ul>

代码效果：

  • 啦啦啦
  • 叭叭叭
  • 嘻嘻嘻
