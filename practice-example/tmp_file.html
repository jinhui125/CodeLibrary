HTML/CSS 知识点汇总

HTML 基础结构

每个HTML文档都应包含以下基本结构：

<!-- 文档类型声明 --> <!DOCTYPE html> <html lang="zh-CN"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>文档标题</title> </head> <body> <!-- 页面内容 --> </body> </html>

常用HTML标签

  • <h1> - <h6>：标题标签，用于定义标题，h1最大，h6最小
  • <p>：段落标签，用于定义文本段落
  • <a>：超链接标签，href属性定义链接地址
  • <img>：图像标签，src属性定义图像路径，alt属性提供替代文本
  • <div>：块级容器，用于布局和样式化
  • <span>：行内容器，用于文本内样式化
  • <ul>, <ol>, <li>：列表标签，分别表示无序列表、有序列表和列表项
  • <table>, <tr>, <td>, <th>：表格标签
  • <form>, <input>, <button>：表单标签

HTML 属性

HTML元素可以拥有属性，属性提供了有关元素的额外信息：

  • id：定义元素的唯一标识
  • class：定义元素的一个或多个类名
  • style：定义元素的内联样式
  • src：定义图像、脚本等外部资源的路径
  • href：定义链接的URL地址
  • alt：定义图像的替代文本
  • data-*：自定义数据属性

CSS 选择器

CSS选择器用于选择要样式化的HTML元素：

/* 元素选择器 */ p { color: blue; } /* 类选择器 */ .className { font-weight: bold; } /* ID选择器 */ #idName { background: yellow; } /* 后代选择器 */ div p { margin: 10px; } /* 伪类选择器 */ a:hover { color: red; }

CSS 盒模型

CSS盒模型包含内容(content)、内边距(padding)、边框(border)和外边距(margin)。

.box { width: 300px; /* 内容宽度 */ height: 200px; /* 内容高度 */ padding: 20px; /* 内边距 */ border: 5px solid gray; /* 边框 */ margin: 10px; /* 外边距 */ box-sizing: border-box; /* 盒模型计算方式 */ }
提示： 设置 box-sizing: border-box; 可以让元素的宽度和高度包含内边距和边框，使布局更直观。

CSS 布局技术

现代CSS提供了多种布局方式：

Flexbox 布局

Flexbox是一种一维布局模型，用于在项目之间分配空间：

.container { display: flex; justify-content: center; /* 水平对齐 */ align-items: center; /* 垂直对齐 */ flex-wrap: wrap; /* 换行 */ }

Grid 布局

Grid是一种二维布局系统，适用于更复杂的布局需求：

.container { display: grid; grid-template-columns: 1fr 1fr 1fr; /* 三列等宽 */ grid-gap: 20px; /* 网格间隙 */ }

CSS 常用属性

  • color：设置文本颜色
  • background：设置背景颜色或图像
  • font-family：设置字体
  • font-size：设置字体大小
  • margin：设置外边距
  • padding：设置内边距
  • border：设置边框
  • display：设置显示方式（block, inline, flex, grid等）
  • position：设置定位方式（static, relative, absolute, fixed）
提示：在实际项目中，使用CSS变量和现代布局技术如Flexbox和Grid可以提高开发效率和代码可维护性。
