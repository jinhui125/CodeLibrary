CSS Grid瀑布流布局

使用CSS Grid和Flexbox创建响应式瀑布流布局，适用于图片展示、博客文章卡片等多种场景

瀑布流布局特点

瀑布流布局（Masonry Layout）是一种流行的网页布局方式，其特点是项目宽度固定，高度不固定，按照一定的规则排列，形成类似瀑布流动的效果。

CSS Grid瀑布流优点

  • 真正的二维布局控制
  • 项目可以轻松跨越行和列
  • 浏览器原生支持，性能好
  • 代码简洁，易于维护

Flexbox瀑布流优点

  • 更好的浏览器兼容性
  • 一维布局控制更直观
  • 项目顺序控制更灵活
  • 在某些简单场景下实现更简单

CSS Grid瀑布流核心代码

.masonry-grid { display: grid; grid-template-columns: repeat(auto-fill, minmax(300px, 1fr)); grid-auto-rows: 10px; /* 控制行高基数 */ grid-gap: 15px; } .masonry-item { /* 项目样式 */ grid-row-end: span X; /* 根据项目高度设置跨越的行数 */ }
