粘性页脚实现指南

粘性页脚（Sticky Footer）是一种网页设计技术，确保页脚始终位于页面底部，即使页面内容不足以填满整个视口高度时也是如此。

传统的页脚设计在内容较少时会在页面中间"漂浮"，而粘性页脚通过CSS技术解决了这个问题。

实现粘性页脚的几种方法

Flexbox方法

使用Flexbox布局是最现代和推荐的方法。将body设置为flex容器，并将main内容区域设置为flex:1，页脚会自动被推到页面底部。

Grid方法

使用CSS Grid布局，将页面划分为网格区域，并将页脚放置在底部区域。

Calc方法

使用CSS calc()函数计算最小高度，确保内容区域至少为视口高度减去页脚高度。

实现代码

以下是使用Flexbox实现粘性页脚的核心CSS代码：

/* 关键CSS代码 */ body { min-height: 100vh; display: flex; flex-direction: column; } main { flex: 1; /* 使主要内容区域占据所有可用空间 */ } footer { margin-top: auto; /* 将页脚推到页面底部 */ }

这种方法利用了Flexbox的弹性布局特性，通过将body设置为flex容器，并让主要内容区域占据所有可用空间，页脚会自动被推到页面底部。

HTML结构

<body> <header> <!-- 页头内容 --> </header> <main> <!-- 主要内容 --> </main> <footer> <!-- 页脚内容 --> </footer> </body>
