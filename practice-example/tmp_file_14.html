JavaScript基础知识点

1. 变量与数据类型

JavaScript中有三种声明变量的方式：var, let, const

// 变量声明示例
var name = "张三"; // 字符串
let age = 25; // 数字
const isStudent = true; // 布尔值

// 数据类型
let number = 42; // 数字
let text = "Hello"; // 字符串
let isTrue = false; // 布尔值
let empty = null; // 空值
let notDefined; // undefined
let person = { // 对象
firstName: "李",
lastName: "四"
};
let colors = ["红", "绿", "蓝"]; // 数组

2. 运算符

JavaScript包含算术运算符、比较运算符和逻辑运算符

// 算术运算符
let a = 10, b = 5;
let sum = a + b; // 15
let difference = a - b; // 5
let product = a * b; // 50
let quotient = a / b; // 2

// 比较运算符
console.log(a == 10); // true
console.log(a === "10"); // false (值和类型都需相等)
console.log(a > b); // true

// 逻辑运算符
let x = true, y = false;
console.log(x && y); // false (与)
console.log(x || y); // true (或)
console.log(!x); // false (非)

3. 条件语句

使用if...else和switch控制程序流程

// if...else语句
let score = 85;
if (score >= 90) {
console.log("优秀");
} else if (score >= 70) {
console.log("良好");
} else {
console.log("需要努力");
}

// switch语句
let day = 3;
switch(day) {
case 1:
console.log("星期一");
break;
case 2:
console.log("星期二");
break;
default:
console.log("其他日子");
}

4. 循环

使用for、while和do...while循环重复执行代码

// for循环
for (let i = 0; i < 5; i++) {
console.log(i); // 输出0,1,2,3,4
}

// while循环
let count = 0;
while (count < 3) {
console.log(count);
count++;
}

// do...while循环
let num = 0;
do {
console.log(num);
num++;
} while (num < 3);

5. 函数

函数是可重复使用的代码块

// 函数声明
function greet(name) {
return "你好, " + name + "!";
}

// 函数调用
let greeting = greet("王五");
console.log(greeting); // "你好, 王五!"

// 箭头函数 (ES6)
const multiply = (a, b) => a * b;
console.log(multiply(4, 5)); // 20

JavaScript练习题

练习1: 变量与数据类型

创建一个包含你个人信息的对象，并显示在页面上。

练习2: 条件语句

编写一个程序，根据输入的成绩判断等级：90+为A，80-89为B，70-79为C，60-69为D，60以下为F。

练习3: 循环

使用循环计算1到100之间所有偶数的和。

练习4: 函数

编写一个函数，接受两个数字参数，返回它们的最大公约数。

练习5: 数组操作

创建一个数组，包含5个你喜欢的颜色，然后：1) 添加一个新颜色，2) 删除第一个颜色，3) 将数组反转。

练习6: 字符串操作

编写一个函数，判断输入的字符串是否是回文（正读反读都一样）。
