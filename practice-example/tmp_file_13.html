SVG常用属性值用法详解

全面介绍SVG图形的基本属性、样式属性和变换属性，包含交互式示例

SVG基本属性

这些属性定义了SVG图形的基本几何特征和位置。

矩形 (rect)

<rect x="20" y="20" width="140" height="80" rx="10" fill="#3498db" stroke="#2c3e50" stroke-width="2"/>

圆形 (circle)

<circle cx="90" cy="60" r="40" fill="#e74c3c" stroke="#c0392b" stroke-width="2"/>

椭圆 (ellipse)

<ellipse cx="90" cy="60" rx="60" ry="30" fill="#2ecc71" stroke="#27ae60" stroke-width="2"/>
属性 描述 常用值 适用元素
x, y 元素左上角坐标 0, 10, 50px rect, text, image
width, height 元素的宽度和高度 100, 50% rect, image, svg
cx, cy 中心点坐标 50, 100 circle, ellipse
r 圆的半径 25, 40px circle
rx, ry 椭圆的x和y半径 40 20 ellipse, rect(圆角)
points 多边形的顶点坐标 0,0 100,0 50,100 polygon, polyline

SVG样式属性

控制SVG图形的外观样式，包括填充、描边、透明度等。

填充颜色 (fill)

描边属性 (stroke)

透明度 (opacity)

属性 描述 常用值 示例
fill 填充颜色 red, #3498db, none fill="#e74c3c"
stroke 描边颜色 blue, #2c3e50 stroke="#3498db"
stroke-width 描边宽度 1, 2px stroke-width="3"
stroke-dasharray 虚线模式 5,5, 10,5 stroke-dasharray="5,5"
opacity 整体透明度 0.5, 1 opacity="0.7"
fill-opacity 填充透明度 0.3 fill-opacity="0.5"
stroke-opacity 描边透明度 0.8 stroke-opacity="0.6"
提示： SVG样式属性既可以直接写在元素上，也可以通过CSS来设置。使用CSS可以更好地维护和复用样式。

SVG变换属性

用于对SVG元素进行平移、旋转、缩放和倾斜等变换操作。

平移 (translate)

transform="translate(50, 30)" /* 向右50px，向下30px */

旋转 (rotate)

transform="rotate(45, 60, 30)" /* 旋转45°，中心点(60,30) */

缩放 (scale)

transform="scale(1.5, 0.8)" /* 水平1.5倍，垂直0.8倍 */
变换函数 描述 语法 示例
translate() 平移变换 translate(tx, ty) translate(50, 30)
rotate() 旋转变换 rotate(angle, cx, cy) rotate(45, 50, 50)
scale() 缩放变换 scale(sx, sy) scale(1.5, 0.8)
skewX() X轴倾斜 skewX(angle) skewX(30)
skewY() Y轴倾斜 skewY(angle) skewY(15)
matrix() 矩阵变换 matrix(a,b,c,d,e,f) matrix(1,0,0,1,50,30)
提示： 多个变换可以组合使用，按从右到左的顺序应用。例如：transform="translate(50,30) rotate(45)" 会先旋转再平移。
