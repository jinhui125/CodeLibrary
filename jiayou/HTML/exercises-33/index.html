瀑布流布局特点

瀑布流布局（Masonry layout）是一种流行的网页布局方式，其特点是项目宽度固定，高度不固定，按照一定的规则排列，形成类似瀑布流动的效果。

CSS Grid瀑布流优点

  • 项目可以轻松跨越行和列
  • 浏览器原生支持，性能好
  • 代码简洁，易于维护
